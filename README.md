

## Функции

- Несколько кортов одновременно
- Синхронизация в реальном времени (Supabase)
- Таймеры и счёт
- Экспорт в Google Sheets
- Работает как PWA (можно установить на телефон)

## Использование

1. Откройте сайт
2. Введите код комнаты (например: `ИГРА1`)
3. Нажмите **Подключиться**
4. Все устройства с одинаковым кодом видят одно и то же
